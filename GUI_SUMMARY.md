## 🎯 **立即使用**

### 📦 **安装和运行**
```bash
# 1. 安装依赖
pip install -r requirements_gui.txt

# 2. 启动应用
python main_gui.py          # 桌面版本
python main_gui.py web       # Web版本
python main_gui.py mobile    # 移动端版本
```

### 🐳 **Docker部署**
```bash
# 构建镜像
docker build -t video-downloader-web .

# 运行容器
docker run -p 8000:8000 video-downloader-web
```

### 🖥️ **Windows快速启动**
```bash
# 使用批处理脚本
build_and_run.bat desktop    # 桌面版本
build_and_run.bat web       # Web版本
build_and_run.bat build     # 构建应用
```
